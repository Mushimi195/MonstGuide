# MonstGuide (モンストガイド)

モンスターストライク（モンスト）用の軌道予測ガイドアプリです。
画面上にオーバーレイ表示され、反射の軌道を描画します。

## ビルドとインストール方法

このプロジェクトは Android Studio プロジェクトとして作成されています。

1.  **Android Studio をインストール**: まだインストールしていない場合は、公式サイトからダウンロードしてインストールしてください。
2.  **プロジェクトを開く**: Android Studio を起動し、"Open" を選択して、この `MonstGuide` フォルダを選択します。
3.  **ビルド**: メニューの `Build` -> `Make Project` を選択します。
4.  **実行**: Android端末をPCに接続（またはエミュレータを使用）し、`Run` ボタン（緑色の再生マーク）を押してアプリをインストールします。

## 使い方

1.  アプリを起動します。
2.  「ガイドを開始」ボタンを押します。
3.  「他のアプリの上に重ねて表示」の権限を求められた場合は、**許可**してください。
4.  許可した後、もう一度「ガイドを開始」を押すと、画面に青い丸と閉じるボタン(X)が表示されます。
5.  **モンストを起動**します。
6.  **青い丸**をドラッグして、自分のキャラクターの位置に合わせます。
7.  画面の適当な場所をドラッグ（引っ張る動作）すると、赤い線で反射の軌道が表示されます。
8.  終了するには、左上の「X」ボタンをタップするか、通知領域から停止してください。

## 注意事項

-   このアプリは物理演算のシミュレーションであり、実際のゲームの挙動と完全に一致するとは限りません。
-   OSのバージョンによっては挙動が異なる場合があります。
